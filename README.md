## 文档

### 1：工具使用

1. 运用了**`Room数据库`**实现了本地数据存储
2. 运用了**`Retrofi`**实现了网络请求和风天气
3. 运用**`Apifox`**管理测试API
4. 运用`ViewModel`管理部分日记，图片，编辑，天气逻辑
5. 运用`bipmap`加载图片
